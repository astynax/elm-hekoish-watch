# Heko'ish Watch

![GitHub](https://img.shields.io/github/license/astynax/elm-hekoish-watch.svg)

highly inspired by awesome [Tokyoflash Heko](http://www.tokyoflash.com/en/watch_museum/tokyoflash/heko).
